# v0.0.1 Core Specification

This document defines and outlines the language's specifications needed for a minimal self-hosted core. The goal of v0.1.0 is to provide the smallest possible primitive set needed to create a compiler and all future higher-level features and layers.

---

## Core Built-Ins

### Types

Two primitive types are provided:
`int // Integer value`
`int* // Pointer to integer`

All other types (arrays, strings, floats, bools) are representable using a combination of `int` and `int*`.

---

### Keywords

The core language defines the following keywords:
`if`
`else`
`while`
`end`

These are enough to express all control flow and function abstractions.

---

### Operators

**Arithmetic**:

`+`, `-`, `*`, `/`

**Comparison**:

`==`, `!=`, `<`, `>`

**Assignment**:

`=`

**Pointer Operations**:

```rust
& // Address of
* // Dereference
```

---

### Literals

The only required literal is Integer:
`0 1 2 3 4 5 ...`

---

## Expressions

The following are supported:

- Arithmetic Expressions
- Comparison Expressions
- Pointer Arithmetic
- Dereference
- Address-Of
- Assignment
- Function Calls

---

## Control Flow

### Conditional

```rust
if (condition) :>
    /* Code */
end
```

---

### Loop

```rust
while (condition) :>
    /* Code */
end
```

---

### Return

```rust
return value
```

---

### Function

#### Function Definition

```rust
type func_name(param: type) :>
    return type
end
```

#### Function Call

```rust
let result: int = add(x, y)
doSmt()
```

---

## Memory Model

All data structures are representable using raw memory and integers.

### Array Indexing

```rust
let elem_ptr: int* = base_ptr + index
let value: int = *elem_ptr
```

---

### Struct Field Access

```rust
let field_ptr: int* = base_ptr + index
let value: int = *field_ptr
```

---

### String Representation (TBI)

```rust
ptr: int* // data
int: int  // length
```

**Minimal Example**:

```rust
int sum(base: int*, len: int) :>
    let i: int = 0
    let s: int = 0
    while (i < len) :>
        s = s + *(base + i)
        i = i + 1
    end

    return s
end
```

---

### Design Goal

v0.0.1 intentioanlly excludes:

- arrays
- structs
- strings
- bool
- modules / imports
- I/O

These will be implemented in the language itself (post v0.0.1)using the core primitives.
